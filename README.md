# 💰 Calculadora de Investimentos

Uma aplicação desenvolvida em React e TypeScript para simular investimentos com juros compostos.

## Funcionalidades

- Simulação de investimentos com juros compostos
- Valor inicial
- Aporte mensal
- Taxa de juros diária, mensal ou anual
- Conversão automática da taxa para o período escolhido
- Cálculo do valor final
- Exibição do total investido
- Exibição dos rendimentos obtidos
- Interface responsiva para desktop e dispositivos móveis
